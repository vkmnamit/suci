Refine and design a modern AI-powered climate intelligence dashboard for smart cities.

The system uses a trained AI model that analyzes urban data (weather, traffic, energy, buildings) and predicts future carbon emissions. It allows users to simulate changes and provides intelligent recommendations.

-----------------------------------
1. MAIN LAYOUT STRUCTURE
-----------------------------------

Create a clean dashboard layout with:

- Left Sidebar (fixed navigation)
- Main Content Area (center)
- Right Analytics Panel
- Bottom Simulation Section (separate, not overlapping)

-----------------------------------
2. SIDEBAR (NEW ADDITION)
-----------------------------------

Add a vertical sidebar on the left:

Include:
- Dashboard (Home)
- Zones / Map
- Forecast
- Scenarios (Simulation)
- Interventions
- Reports
- Settings

Design:
- Minimal icons + labels
- Active tab highlight
- Smooth hover animation
- Glassmorphism / soft blur effect

-----------------------------------
3. HOME DASHBOARD (MAIN PAGE)
-----------------------------------

Top section:
- Global stats cards:
  - Total Carbon Emission
  - Energy Usage
  - Active Zones
  - Renewable %

Center:
- Large city map (hero section)
- Zones clearly visible (no overlap)
- Grid-aligned placement (structured, not random)

Left floating panel:
- Zone overview
- Alerts (pollution warnings)

Right panel:
- Forecast charts
- Carbon trend
- Model performance

-----------------------------------
4. MAP IMPROVEMENTS (IMPORTANT)
-----------------------------------

- Ensure all zones (Rohini, Connaught, Noida, etc.) are fully visible
- No overlapping with panels
- Add proper spacing between zones
- Add padding around edges
- Use grid or structured layout
- Subtle glow for active zones
- Labels must be fully readable

-----------------------------------
5. SIMULATION SECTION (BOTTOM)
-----------------------------------

- Separate section below map (not overlapping)
- Sliders:
  - Traffic
  - Energy
  - Solar %
- “Run Simulation” button
- Before vs After comparison
- Show carbon reduction clearly

-----------------------------------
6. AI RECOMMENDATION PANEL
-----------------------------------

- Cards with:
  - Suggestion (e.g., increase solar)
  - Impact level (High / Medium)
  - Percentage improvement
- Ranked list
- Clean card UI with soft shadows

-----------------------------------
7. RESPONSIVENESS (VERY IMPORTANT)
-----------------------------------

- Layout should adapt to different screen sizes
- Sidebar collapses on smaller screens
- Panels stack properly (no overlap)
- Maintain spacing and readability
- Grid-based responsive layout

-----------------------------------
8. ANIMATIONS (PROFESSIONAL 🔥)
-----------------------------------

Add subtle and premium animations:

- Smooth hover effects on cards
- Sidebar expand/collapse animation
- Chart loading animations
- Map zone highlight animation
- Button click feedback
- Transition between states (before/after simulation)

Keep animations:
- Fast
- Smooth
- Not flashy (professional SaaS feel)

-----------------------------------
9. COLOR THEME
-----------------------------------

Primary colors:
- Black (#0B0B0B)
- White (#FFFFFF)
- Beige (#E8DCCF)

Usage:
- Background → black
- Cards → dark with slight transparency
- Highlights → beige
- Alerts → soft red tones

-----------------------------------
10. STYLE & FEEL
-----------------------------------

- Minimal, clean SaaS dashboard
- Glassmorphism cards
- Soft shadows
- Modern typography
- Premium, futuristic but simple

Inspiration:
- Tesla dashboard
- Smart city control panels
- High-end analytics tools

-----------------------------------
FINAL GOAL
-----------------------------------

The UI should look like a real production-ready AI SaaS product that:
- Visualizes city data
- Predicts future outcomes
- Allows simulation
- Provides intelligent decisions
